Theme 1: Basic web

Responsive website


Individual work


[Communication and presentation] [Design and visualisation]


For this task I had a to code a responsive website about a certain art style, I was given pop art. I learned about gestalt laws which taught me how to create an aesthetically pleasing website with symmetry and balance. It was my first time using HTML and CSS, I learned how to create grids in CSS, which was challenging yet manageable. It was also my first time using Photoshop and Illustrator to create my website’s logo.
